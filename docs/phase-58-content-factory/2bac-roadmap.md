# 2BAC Roadmap

This roadmap defines the first production direction for 2BAC content expansion in OQUL.

It is a planning document, not a generation script.

## Mathematics

Estimated total: 40-60 high-quality lessons.

### Modules

1. Functions
2. Limits
3. Continuity
4. Derivatives
5. Variations and extrema
6. Exponential function
7. Logarithmic function
8. Integrals
9. Sequences
10. Complex numbers
11. Probability
12. Geometry in space

### Priority Order

1. Derivatives and variations
2. Exponential and logarithmic functions
3. Limits and continuity
4. Integrals
5. Complex numbers
6. Sequences
7. Probability
8. Geometry

### First Pilot Lessons

1. Derivative definition
2. Derivative rules
3. Variations and extrema
4. Exponential function
5. Logarithmic function

## Expansion Principle

Do not generate all roadmap lessons at once.

Start with one lesson, then five lessons, then one module, then expand by subject.
