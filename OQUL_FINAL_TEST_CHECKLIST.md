# Oqul Final Test Checklist

After `npm run start`, open these pages:

1. `/` landing page: buttons go to register/login/demo pages.
2. `/register`: create a new account.
3. `/onboarding`: add a child and choose a grade.
4. `/dashboard/lessons`: lessons should appear automatically. If no child exists, it still shows grade/subject/lesson browsing.
5. Open any lesson: explanation and exercises should load.
6. `/dashboard/leila`: ask Leila a question.
7. `/test-routes`: public route smoke test.

Important: run `db-setup.ps1` once before testing lessons.
